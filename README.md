
# Linux version of the justvpn client.

![screen](https://user-images.githubusercontent.com/25012104/155532524-98389838-34b2-4377-8191-75320a2f6b1a.png)

## BUILD

* git clone https://github.com/gantzzz/justvpndesktop.git
* cd justvpndesktop
* mkdir build
* cd build
* cmake ../
* make
